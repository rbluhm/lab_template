# sandbox/

Scratch space for agents (and humans). **Everything here is git-ignored.**

Agents are free to write, rename, and delete files in this folder without
asking: exploratory scripts, intermediate dumps, draft figures, throwaway
analyses, notes-to-self. Nothing here is part of the replication package.

## Promotion

A file leaves the sandbox only when a **human says "promote it"**. On promotion
it is moved (not copied) into its real home and cleaned up to project standard:

- reusable build/cleaning code  → `code/build/`
- processing / transformation   → `code/process/`
- analysis & results code       → `code/analyze/`
- a written-up finding or memo   → `notes/`

Because this folder is ignored, do not rely on it for anything you want to keep.
If it matters, promote it.
